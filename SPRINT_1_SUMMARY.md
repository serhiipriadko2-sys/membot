## Инструкции по развёртыванию

### 1. Обновите ключ Supabase

```bash
# Откройте .env и замените placeholder на действительный ключ
nano .env
# SUPABASE_KEY=sb_secret_YOUR_ACTUAL_KEY_HERE
```

### 2. Примените миграции

**Вариант A: Через Dashboard (рекомендуется)**
1. Перейдите в https://app.supabase.com
2. Выберите проект `catntmobfcenkhkodnqv`
3. SQL Editor → вставьте содержимое `migrations/001_initial_schema.sql`
4. Нажмите Run

**Вариант B: Через CLI**
```bash
npx supabase db push
```

### 3. Проверьте подключение

```bash
python src/storage/supabase_client.py
```

Ожидаемый вывод:
```
[FACT] Connected to Supabase project: catntmobfcenkhkodnqv
✓ Supabase connection successful
```

### 4. Запустите тесты

```bash
make test
# или
python -m pytest tests/ -v
```
