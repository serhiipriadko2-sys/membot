# Iskra Signal Forge UI Style v2

## Visual SoT
The visual anchor is the generated concept board: obsidian forensic UI, spark-gold fracture, cyan/violet signal glow, glass panels, dense but breathable data cards.

## Brand law
Interface is not decoration. It must lead to action: raw evidence, replay, hypothesis, and verification.

## Palette
- Spark Gold: `#FFC36B`
- Ember Orange: `#FF6A1A`
- Plasma Violet: `#7A5CFF`
- Signal Cyan: `#00E6FF`
- Mint Proof: `#00FFC2`
- Slate Panel: `#1A1F2B`
- Obsidian Void: `#0B0E14`

## Signature motifs
- Vertical sigil / spark mark.
- Diagonal golden fracture seam.
- Glass panels with thin amber/cyan borders.
- Truth Ladder and evidence chips.
- Terminal-like signature trace.
- Right intelligence rail with hypothesis note and cluster map.

## Guardrails
- Beauty must not imply raw truth.
- Missing artifacts render UNKNOWN, not confidence.
- Pre-buy triggers remain hypotheses until raw replay and controls pass.
- No trading actions in the UI.

## QA checklist
- First screen should feel like Iskra: mythic, forensic, expensive, technological.
- At least one visible golden spark seam.
- Truth Ladder visible before detailed tables.
- Artifact coverage visible before claims.
- Mobile layout collapses without hiding Upload.
